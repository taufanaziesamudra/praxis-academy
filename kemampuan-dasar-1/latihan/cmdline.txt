cd "nama file/folder => untuk mengeksekusi file/folder.
cd.. => kembali ke file atau folder sebelumnya.
mkdir "nama folder" => untuk membuat folder.
chdir => untuk mengetahui folder yang sedang aktif. 
dir => untuk meilhat keseluran file atau folder pada direktori tertentu.
systeminfo => untuk menampilkan informasi komputer yang digunakan.
tree => untuk menampilkan struktur dalam suatu folder.
tasklist => untuk menampilkan informasi aplikasi yang sedang berjalan di komputer.
help => untuk menampilkan fungsi-fungsi apa saja yang bisa dipakai.
driverquery => untuk melihat daftar apa saja yang terinstal pada driver komputer.
cls => untuk mengkosongkan command prompt.
date => untuk menampilkan tanggal pada layar cmd.
start => untuk membuka jendela CMD baru.
ftype => untuk menampilkan tipe file beserta informasi aplikasi yang bisa digunakan untuk membukanya.
color => untuk mengganti tema warna
rename => untuk digunakan mengubah nama file dan direktori.
del => untuk menghapus file.
copy => untuk menyalin file kelokasi yang berbeda.
chkdsk => untuk memeriksa status hard drive.
logoff => untuk perintah keluara dari komputer.
